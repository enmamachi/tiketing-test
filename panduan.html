Aplikasi ticketing ini kompatibel dengan berbagai versi XAMPP:

Versi XAMPP Versi PHP Status Kompatibilitas Rekomendasi
XAMPP 8.2.x PHP 8.2 Kompatibel ✅
XAMPP 8.1.x PHP 8.1 Kompatibel ✅
XAMPP 8.0.x PHP 8.0 Kompatibel ✅
XAMPP 7.4.x PHP 7.4 Kompatibel ✅

Catatan: Direkomendasikan menggunakan XAMPP 7.4.x atau yang lebih baru untuk kompatibilitas dan keamanan terbaik.

Langkah Instalasi

1Download dan Install XAMPP

Download XAMPP dari website resmi:

https://www.apachefriends.org/download.html

Pilih versi yang sesuai dengan sistem operasi Anda (Windows, Linux, atau macOS).

2Jalankan XAMPP Control Panel

Setelah instalasi selesai, buka XAMPP Control Panel dan start modul Apache dan MySQL.

# Klik tombol 'Start' untuk Apache
# Klik tombol 'Start' untuk MySQL

3Persiapkan Database

Buka browser dan akses http://localhost/phpmyadmin

Buat database baru dengan nama ticketing_system

CREATE DATABASE ticketing_system;

4Letakkan File Aplikasi

Simpan semua file aplikasi ticketing ke folder:

C:\xampp\htdocs\ticketing-system (Windows)
/opt/lampp/htdocs/ticketing-system (Linux)
/Applications/XAMPP/htdocs/ticketing-system (macOS)

5Konfigurasi Database

Buat file config.php di folder aplikasi dengan kode berikut:

<?php
$host = 'localhost';
$dbname = 'ticketing_system';
$username = 'root';
$password = '';

try {
    $pdo = new PDO("mysql:host=$host;dbname=$dbname", $username, $password);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
} catch (PDOException $e) {
    die("Koneksi database gagal: " . $e->getMessage());
}
?>

6Buat Tabel Database

Jalankan query berikut di phpMyAdmin untuk membuat tabel tickets:

CREATE TABLE tickets (
    id INT(11) AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL,
    branch VARCHAR(50) NOT NULL,
    unit VARCHAR(20) NOT NULL,
    issue_type VARCHAR(30) NOT NULL,
    priority VARCHAR(10) NOT NULL,
    subject VARCHAR(200) NOT NULL,
    description TEXT NOT NULL,
    status VARCHAR(20) DEFAULT 'Open',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

7Akses Aplikasi

Buka browser dan akses:

http://localhost/ticketing-system

Aplikasi ticketing siap digunakan!

Konfigurasi Email (Opsional)

Untuk mengaktifkan fitur notifikasi email, ikuti langkah berikut:

1Konfigurasi PHP.ini

Buka file php.ini di folder XAMPP (biasanya di C:\xampp\php\php.ini)

Cari dan modifikasi pengaturan berikut:

[mail function]
SMTP = smtp.gmail.com
smtp_port = 587
sendmail_from = your.email@gmail.com
sendmail_path = "\"C:\xampp\sendmail\sendmail.exe\" -t"

2Konfigurasi Sendmail

Buka file sendmail.ini di folder C:\xampp\sendmail\sendmail.ini

Ubah pengaturan sesuai dengan provider email Anda:

smtp_server=smtp.gmail.com
smtp_port=587
error_logfile=error.log
debug_logfile=debug.log
auth_username=your.email@gmail.com
auth_password=your-app-password
force_sender=your.email@gmail.com

Peringatan: Untuk Gmail, Anda perlu membuat "App Password" karena password biasa tidak akan bekerja. Matikan juga verifikasi 2 langkah sementara saat konfigurasi.

Pemecahan Masalah (Troubleshooting)

Apache tidak bisa start

Jika Apache tidak bisa dijalankan, kemungkinan port 80 atau 443 sedang digunakan.

Solusi: Ganti port Apache di httpd.conf dan httpd-ssl.conf

Error koneksi database

Pastikan MySQL berjalan dan kredensial database di config.php benar.

Email tidak terkirim

Periksa konfigurasi sendmail.ini dan pastikan menggunakan app password untuk Gmail.
